## Project Description

This project is based on the research topic:

Digital Ghosting and Psychological Stress: Exploring AI-Mediated Social Rejection.

The application analyzes emotional stress caused when AI systems suddenly stop replying or become unavailable during interaction.

## Features

- Stress Analyzer
- Dashboard Visualization
- User Friendly Interface
- Research Based Prototype
- Practical Implementation of Research Concept

## How to Run

1. Install Python 3.10 or above.

2. Open terminal in project folder.

3. Install dependencies:

pip install -r requirements.txt

4. Run the project:

python -m streamlit run app.py

5. Open browser and visit:

http://localhost:8501

## Expected Output

The system accepts user emotional responses and predicts stress level as:

- Low Stress  
- Medium Stress  
- High Stress  

It also displays dashboard statistics through charts.
